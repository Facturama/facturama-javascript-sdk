Ejemplos de API Web

El API en su modalidad Web , permite el timbrado desde una única Entidad fiscal (un único RFC),
Todos los CFDIs creados en esta modalidad, son visibles y consultables mediante la plataforma web
Los datos de la entidad fiscal (Nombre, RFC, Certificado, dirección, etc) se configuran desde la plataforma de Facturama
en la sección de Perfil fiscal.

Ejemplo de CRUD de clientes

Para poder generar un CFDI es necesario que tengas registrado previamente el cliente
Los datos de los clientes, pueden ser modificados también desde la plataforma de Facturama Catálogo de clientes

Listar clientes

Parametros utlizados:

  • start - Indice del registro - Valores permitidos de tipo int(0 .. n < = 100)
  • lenght - tamaño de registro mostrado - Valores permitidos de tipo int(1 .. 100)
  • search - Valores permitidos de tipo string()
start = 0 & length = 100 & search = null

Los datos de los clientes, pueden ser modificados también desde la plataforma de Facturama Catálogo de clientes
